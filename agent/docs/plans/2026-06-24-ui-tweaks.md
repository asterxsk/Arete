### Task 2: Truncate args to 50 chars in compactui

**Files:**
- Modify: `C:/Users/prithish/.pi/agent/extensions/compactui/index.ts:50`

The `compactCall` function passes `argsStr` directly. Truncate it to 50 chars max.

- [ ] **Step 1: Add truncation to compactCall**

Find the `compactCall` function:

```ts
function compactCall(toolName: string, argsStr: string, theme: any): Component {
	return line(INDENT + orange(theme, toolName) + " [" + argsStr + "]" + theme.fg("dim", HINT));
}
```

Change to:

```ts
function compactCall(toolName: string, argsStr: string, theme: any): Component {
	const display = argsStr.length > 50 ? argsStr.slice(0, 47) + "..." : argsStr;
	return line(INDENT + orange(theme, toolName) + " [" + display + "]" + theme.fg("dim", HINT));
}
```

- [ ] **Step 2: Verify**

Reload pi. A bash call with a long command should show:
```
 bash [git log --oneline --graph --decorate --all...] (ctrl+o to expand)
```
truncated at 50 chars with `...`.

**Files:**
- Modify: `C:/Users/prithish/.pi/agent/extensions/subagents/index.ts` (multiple render sites)

- [ ] **Step 1: Fix the running-state render (line ~1039)**

Find the render for when results are still running:


```ts
return new CompactToolBox({
	toolName: agentName,
```
